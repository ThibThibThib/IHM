Introduction

Une Interaction Homme Machine permet à un utilisateur d'interagir avec un ordinateur, une tablette, un smartphone.
La souris et le clavier, l'écran tactile permettent desIHM.
Nous allons étudier ici les IMH sur une page WEB codée en HTML, nous aborderons le CSS et le JavaScript qui permettent de créer une grande richesse d'IHM.
